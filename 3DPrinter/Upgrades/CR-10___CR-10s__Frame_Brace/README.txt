# Summary

Yellow,

I wanted a slim, minimal frame brace design for my CR-10.
I found Rick’s (RickB) design, which i really liked, but it’s in inches and i found it a a little bit difficult to install since he design sockets for the nuts in both ends. 
So i design my frame stiffeners in Fusion 360 from scratch in metric, based on RickB design.

You will need:
  2 x (M10 all-thread x 61.5cm long)
  8 x (M10 hex nuts)
16 x ( M5 x 8mm long) (up to 10mm long may work fine)
16 x ( M5 T-nuts)
  4 x ( Printed parts)
